# 題⽬ M：經典⾵險因⼦模型之⽐較與解釋⼒分析

## 內容說明
本專案旨在選擇至少十檔上市公司股票，整理其基本特徵，並分析三種經典資產定價模型（CAPM、Fama-French 三因子、Fama-French 五因子）的理論架構、因子來源與可解釋的風險。最後，將比較三種模型在理論上的相對解釋力。

---

## 專案要求

### 1. 選股與特徵整理
- 選擇至少 10 檔股票
- 整理每檔股票的基本特徵（如產業、規模、估值）

### 2. 模型介紹
- 介紹 CAPM、Fama-French 三因子、Fama-French 五因子模型

### 3. 模型比較與差異
- 每個新增因子解決了什麼問題？
- 哪些股票特徵是 CAPM 無法解釋、但 FF3 或 FF5 可以解釋的？

### 4. 理論解釋力分析
- 你的投資組合中，哪些股票較符合 CAPM？哪些嚴重偏離？
- FF3 模型是否因 SMB/HML 捕捉到更多風險？
- FF5 模型是否因 RMW/CMA 更全面？

### 5. 完整報告
- 撰寫完整報告，包含圖表、數據分析與結論
