\section{模型架構}
本研究比較三種經典資產定價模型。以 Fama-French 五因子模型為例，其回歸方程式如下：
\begin{equation}
R_{it} - R_{ft} = \alpha_i + \beta_i(R_{Mt} - R_{ft}) + s_i SMB_t + h_i HML_t + r_i RMW_t + c_i CMA_t + \epsilon_{it}
\end{equation}
